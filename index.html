About This Project

PhishGuard is a machine learning–based phishing detection system designed to analyze URLs and identify phishing attacks in real time.

The model evaluates 48 handcrafted security features including URL length, entropy, domain structure, sensitive keywords, impersonation patterns, and protocol indicators.

This project demonstrates how machine learning can be applied to modern cybersecurity challenges.
