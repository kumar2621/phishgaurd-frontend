About This Project

PhishGuard is a machine learning–based phishing detection system designed to analyze URLs and identify potential phishing attacks in real time.

The model uses 48 handcrafted features including URL length, entropy, domain structure, sensitive keywords, impersonation patterns, and protocol indicators.

This project demonstrates how machine learning can be effectively applied to cybersecurity.
