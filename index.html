About This Project

PhishGuard is a machine learning–based phishing detection system designed to analyze URLs and identify potential phishing attacks in real time. Instead of relying on traditional blacklists, the system extracts meaningful features from a given URL and evaluates its structural and behavioral patterns.

The model uses 48 handcrafted features including URL length, entropy, domain structure, presence of sensitive keywords, brand impersonation, and protocol-level indicators. These features are passed into a trained classification model to determine whether a URL is legitimate or potentially malicious.

This project demonstrates how machine learning can be effectively applied to cybersecurity for early detection and prevention of phishing attacks.
